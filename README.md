## To Run

Install necessary dependencies and run, each in a separate termial and port
```
python ./fb_module/api.py # runs on port 5000
```
```
python -m fb_module
```
```
python ./scraper/api.py # runs on port 5002
```
```
python -m scraper
```
```
python -m ./cv_module/api.py # runs on port 5001
```
